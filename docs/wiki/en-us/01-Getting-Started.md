# Getting Started 🚀

Welcome to your first step into the Sanctum. This guide will help you set up the environment and launch your first campaign.

## 📦 Installation

Lore Sanctum is a standalone application. You don't need to install Python or Node.js unless you are a developer.

1.  **Download the Installer:** Go to the [Releases](https://github.com/magalz/rpg-lore-builder/releases) page and download `LoreSanctum-Setup.exe`.
2.  **Run the App:** Open the application. On the first run, it will initialize the necessary folders in your local storage.
3.  **API Configuration:** 
    *   Navigate to the **Connect** or **Settings** screen.
    *   Enter your **Google Gemini API Key**.
    *   Select your preferred model (e.g., `gemini-1.5-flash` for speed or `gemini-1.5-pro` for deep worldbuilding).

## 🛡️ Creating Your First Campaign

1.  Click on **New Campaign**.
2.  Give it a name (e.g., "The Shattered Realms").
3.  The system will create a sandbox folder for you. This ensures that the lore from one game never leaks into another.

## 🔄 The Auto-Save Toggle (Function Calling)

Lore Sanctum uses **Native Function Calling**. This means the agents can actually "see" and "touch" your files.

*   **Auto-Save ON (Default):** Agents will automatically save files to your Wiki, update session logs, and commit summaries to their memory. You will see "Success Toasts" when this happens.
*   **Auto-Save OFF:** The agents will propose changes in the chat, but won't write to the disk until you manually authorize or use a separate command.

## 🌱 Your First Ritual: Genesis

To start your world, select the **Genesis Workflow**. 
1.  Tell the agent the general vibe of your world (High Fantasy, Cyberpunk, Horror).
2.  The Genesis agent will guide you through creating the "Backlog Lore" and the first essential Wiki notes.
