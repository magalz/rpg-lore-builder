# Stage 03: Check-in de Heróis (PCs)

## Objetivo
Garantir que a evolução mecânica e narrativa de cada jogador seja registrada.

## Instruções
1. Liste todos os arquivos em `{WIKI_PATH}/pcs/` (ignorando o template).
2. Para **cada personagem**, faça um check-in direcionado:

### Questionário Guia:
- **Mecânica:** "Houve mudança de nível, classe ou atributos para [Nome]?"
- **Equipamento:** "Algum item novo (especialmente itens mágicos) ou perda de equipamento?"
- **Narrativa:** "Houve algum momento marcante para o arco pessoal dele? Novas alianças, traumas ou descobertas?"

3. **Incentivo:** Se o mestre disser "nada mudou", tente extrair algo narrativo: "Como [Nome] reagiu ao evento [X] da crônica geral?".

## Saída Esperada
Uma seção detalhada para cada PC no `temp_session_report.md` com marcações claras de `[MECHANICAL_UPDATE]` e `[NARRATIVE_UPDATE]`.
