# Stage 03: Auditoria de Cânone (The Inquisitor)

## Instruções
1. Envie o roteiro completo (Narrativa + Tática) para o **rlb-agent-inquisitor**.
2. Use a capacidade `validate-entry` do Inquisitor.
3. O Inquisitor deve verificar:
   - Os inimigos escolhidos pelo Tactician fazem sentido na região (Lore)?

4. Se houver conflitos, o Inquisitor deve sugerir correções imediatas.

## Saída Esperada
