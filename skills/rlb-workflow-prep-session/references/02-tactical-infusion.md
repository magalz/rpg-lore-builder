# Stage 02: Infusão Tática (The Tactician)

## Instruções
1. Identifique todas as instâncias da tag `[ENCONTRO: Dificuldade, Tema]` no esboço do Weaver.
2. Para cada tag encontrada:
   - Invoque o **rlb-agent-tactician**.
   - Use a capacidade `evaluate-party` para ler os PCs em `wiki/pcs/`.
   - Use a capacidade `build-encounter` para gerar a matemática e tática para aquele encontro específico.
   - Se o monstro não estiver na wiki, peça ao Tactician para usar `fetch-stats`.

3. **Substituição:** Troque a tag `[ENCONTRO]` pelo bloco de estatísticas e táticas gerado pelo Tactician.

## Saída Esperada
O roteiro narrativo agora enriquecido com dados táticos e mecânicos precisos.
